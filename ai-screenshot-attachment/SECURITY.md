# Security Policy

## Supported versions

This project is early-stage. Security fixes will generally land on the latest version.

## Reporting a vulnerability

Please do not open a public issue for sensitive vulnerabilities.

Instead, contact the maintainer privately with:
- a short summary
- impact
- reproduction steps
- affected macOS version
- affected Python environment details if relevant

You should receive an acknowledgment and follow-up once the report is reviewed.
