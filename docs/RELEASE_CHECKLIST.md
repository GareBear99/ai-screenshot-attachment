# Release Checklist

## Before tagging

- [ ] Validate the README examples
- [ ] Test capture with Safari and one Chromium-based browser
- [ ] Confirm final stdout line is the image path
- [ ] Update changelog
- [ ] Review issue templates and support docs

## Tag format

Use semantic version tags:

```text
v0.1.0
```

## Suggested first release notes

- Initial public release
- Default browser detection
- Specific-app targeting
- Optional URL opening
- Window capture fallback behavior
